# 🚀 Full-Stack Web Applications

A monorepo of production-ready full-stack web applications built with a consistent modern JavaScript stack — React (Vite), Node.js, Express, and MongoDB — all deployed on Vercel.

> Built to demonstrate end-to-end ownership: from UI design and component architecture to RESTful APIs, database modeling, and cloud deployment.

## 🛍️ UrbanWear — Fashion E-Commerce

A fully functional online store for fashion products, built to mirror the core experience of a real-world e-commerce platform.

**Features**

- 🛒 Shopping cart with real-time quantity and total updates
- ❤️ Wishlist to save and manage favourite products
- 💳 Checkout flow with order summary
- 📍 Address management (add, edit, select delivery address)
- 🔐 User authentication (JWT-based)

**Tech Stack**

`React` `Vite` `Node.js` `Express.js` `MongoDB` `Mongoose` `JWT` `Vercel`

---

## 📊 Anvaya CRM — Lead Management

A B2B CRM tool designed to help sales teams track and manage leads through a structured pipeline, with built-in reporting and activity logging.

**Features**

- 🔄 5-stage lead pipeline (visual board-style progression)
- 🔍 Filtering and search across leads
- 💬 Activity comments per lead for team collaboration
- 📈 Chart.js-powered analytics dashboard with visual reports
- ☁️ Deployed and live on Vercel

**Tech Stack**

`React` `Vite` `Chart.js` `Node.js` `Express.js` `MongoDB` `Mongoose` `Vercel`

---

## 🗓️ Meetup App — Community & Events

A community-driven event platform where users can browse, create, and manage meetups and local group events.

**Features**

- 📅 Event listing and detail views
- 👥 Group and community management
- 🔐 User authentication and session handling

**Tech Stack**

`React` `Vite` `Node.js` `Express.js` `MongoDB` `Vercel`

## ✅ Workasana — Project Management

A task and project management application inspired by modern productivity tools, built to help teams organize work, assign tasks, and track progress.

**Features**

- 📁 Project and task organization
- 🏷️ Task assignment and status tracking

## 🧰 Common Tech Stack

All applications in this repo share a consistent, production-grade JavaScript stack:

| Layer               | Technology                                |
| ------------------- | ----------------------------------------- |
| **Frontend**        | React 18, Vite, CSS Modules / Vanilla CSS |
| **Backend**         | Node.js, Express.js                       |
| **Database**        | MongoDB, Mongoose ODM                     |
| **Auth**            | JSON Web Tokens (JWT)                     |
| **Charts**          | Chart.js (Anvaya CRM)                     |
| **Deployment**      | Vercel (frontend + serverless backend)    |
| **Version Control** | Git, GitHub                               |
